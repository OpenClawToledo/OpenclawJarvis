# 📋 Checklist de Onboarding — Novo Cliente

## 1. Dados a coletar do cliente
- [ ] Nome do negócio
- [ ] Nicho/tipo de produto ou serviço
- [ ] Instagram (usuário)
- [ ] WhatsApp Business (número)
- [ ] Responsável (nome + contato)
- [ ] Horário de funcionamento
- [ ] Cidade/bairro (para negócios locais)
- [ ] Produtos/serviços principais + preços
- [ ] Identidade visual (cores, logo)?
- [ ] Meta mensal de faturamento

## 2. Setup técnico
- [ ] Criar pasta /agencia/{cliente}/
- [ ] Criar PLANO.md com estratégia
- [ ] Criar legendas semana 1 e 2
- [ ] Configurar crons de conteúdo (Haiku)
- [ ] Testar envio de mensagem

## 3. Setup do cliente
- [ ] Configurar WhatsApp Business (mensagem automática)
- [ ] Criar catálogo no WhatsApp Business
- [ ] Otimizar bio do Instagram
- [ ] Link na bio apontando pro WhatsApp

## 4. Contrato / acordo
- [ ] Valor mensal definido
- [ ] Forma de pagamento
- [ ] O que está incluído
- [ ] Prazo mínimo (recomendado: 3 meses)
