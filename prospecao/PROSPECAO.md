## 📋 PIPELINE DE VENDAS

```
NOVO → CONTACTADO → PROPOSTA → NEGOCIAÇÃO → FECHADO ✅
                                           → PERDIDO ❌
```

### Definições de cada estado:
| Estado | Significado |
|---|---|
| **NOVO** | Lead identificado, ainda não contactado |
| **CONTACTADO** | Primeira mensagem enviada, aguarda resposta |
| **PROPOSTA** | Proposta enviada, cliente a analisar |
| **NEGOCIAÇÃO** | Há interesse, ajustes de valor/âmbito |
| **FECHADO** | Contrato assinado ou acordo verbal + data |
| **PERDIDO** | Não quer avançar (anotar motivo) |

## 🔍 MÉTODOS DE PROSPEÇÃO

### Método 1 — Google Maps Hunting (mais eficaz)
1. Abrir Google Maps
2. Pesquisar "restaurante [cidade]" / "cabeleireiro [cidade]" / etc.
3. Filtrar por: sem site (aparece "Sem website") OU site básico/antigo
4. Copiar nome, telefone, rua → adicionar como LEAD no painel
5. Enviar mensagem via WhatsApp do negócio (se disponível)

**Meta:** 10 leads por hora de pesquisa

### Método 2 — Instagram por Localização
1. Pesquisar por localização no Instagram
2. Ver posts de negócios locais
3. Verificar se têm site na bio → se não têm = oportunidade
4. DM directo pelo Instagram ou comentário estratégico

### Método 3 — Referências (o mais rentável)
1. Cada cliente entregue = pedir 2 contactos
2. "Conhece alguém que precisaria de algo assim?"
3. Oferecer desconto de €50 por cada referência que feche
4. **Jiaxin Song** = porta de entrada para comunidade chinesa

### Método 4 — Passeio Comercial (offline)
1. Toledo vai a pé pela rua comercial
2. Entra nos negócios sem site visível
3. Apresenta-se com cartão ou QR code
4. Convida para 10 min de conversa

### Método 5 — LinkedIn (gabinetes e profissionais)
1. Pesquisar "contabilista [cidade]" / "advogado [cidade]"
2. Ver perfil → sem site pessoal ou empresa = oportunidade
3. Mensagem de conexão + proposta de valor
