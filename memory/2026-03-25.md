### Lições aprendidas
- **Nunca editar ficheiros gerados pelo Vite** em `src/main/resources/static/` directamente — o próximo build sobrescreve
- **Sempre editar `frontend/index.html`** para meta tags, theme-color, etc.
- **Imports de imagens públicas em Vue**: usar `:src="computed"` ou variável `data()` — não `src="/caminho"` (Rollup tenta resolver)
- **`emptyOutDir: false` no vite.config.js** acumula ficheiros CSS antigos — não afecta funcionalidade mas aumenta tamanho do JAR
- **`position: fixed` no nav mobile** causava conflito com z-index e scroll — usar `position: absolute` dentro do header fixed
